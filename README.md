# Christus Luo Portfolio

Static portfolio site for `christusluo.com`.

## Pages

- `index.html` - home page and visual introduction
- `work.html` - three selected work/project cards
- `about.html` - personal background and working style
- `resume.html` - resume summary and linked PDFs
- `playground.html` - vibe coding and experimental media shelf
- `404.html` - fallback page for static hosting

## Deployment

This site is plain HTML, CSS, and JavaScript. It can be deployed with GitHub Pages, Netlify, Vercel, or any static host.

For GitHub Pages, keep `CNAME` in the published root so the custom domain resolves to `christusluo.com`.

## Editing Content

- Update the project cards in `work.html`.
- Update bio copy in `about.html`.
- Update resume links and timeline entries in `resume.html`.
- Add small experiments to the `.play-grid` section in `playground.html`.
- Shared styles live in `styles.css`.
